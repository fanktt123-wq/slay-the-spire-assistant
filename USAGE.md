# 使用指南

## 启动

双击 `start.bat`，然后访问 http://localhost:3000

## 三个功能

### 1. 实时对战 (/battle)

配置战斗场景，获取 AI 决策建议：
- 点击角色面板：编辑生命值、金币、遗物、药水
- 点击"添加敌人"：添加战斗中的敌人
- 点击手牌区域：编辑当前手牌
- 点击抽牌堆/弃牌堆/消耗堆：管理牌堆
- 点击"AI 决策"：获取 AI 战斗建议
- 点击"保存"/"加载"：保存战斗状态

### 2. 卡组构筑 (/deck)

构建卡组并获取 AI 分析：
- 点击角色按钮切换角色
- 在右侧卡牌库点击卡牌添加到卡组
- 在遗物区域点击遗物进行选择
- 在底部聊天框输入问题，AI 会分析你的构筑
- 点击 📚 按钮选择要发送给 AI 的知识库

### 3. 数据管理 (/admin)

管理游戏数据：
- 访问 http://localhost:3000/admin
- 可以管理：角色、卡牌、遗物、Buff、敌人、药水
- 支持从 JSON 导入数据

## 配置 LLM API

1. 打开卡组构筑或实时对战页面
2. 点击右上角的 ⚙️ 按钮
3. 选择预设：
   - **Kimi** (推荐国内用户)
   - **Claude** (推荐国外用户)
   - **OpenAI**
   - **自定义**
4. 填写 API 密钥

## 常见问题

**Q: API 调用失败？**
- 检查 API 密钥是否正确
- 检查网络连接
- 确认账户有足够额度

**Q: 如何更新游戏数据？**
- 访问数据管理界面 (/admin)
- 或者编辑 data/ 目录下的 JSON 文件后重启服务
