# 启动说明

## 最简单的启动方式

**双击运行：**
```
start.bat
```

然后浏览器会自动打开 http://localhost:3000

在页面上点击你想用的功能：
- ⚔️ 实时对战
- 🎯 卡组构筑  
- 🗃️ 数据管理

## 配置 AI

首次使用需要配置 API：
1. 进入卡组构筑或实时对战页面
2. 点击右上角的 ⚙️ 按钮
3. 选择 AI 服务（Kimi、Claude、OpenAI）
4. 填写你的 API 密钥

## 故障排除

**如果端口被占用**
编辑 `backend/.env`：
```
PORT=3001
```

**如果提示缺少 Node.js**
访问 https://nodejs.org/ 下载安装
