# 快速参考

## 启动

```bash
start.bat
```

访问 http://localhost:3000

## API 配置示例

| 服务商 | API 地址 | 模型 |
|--------|----------|------|
| Kimi | https://api.moonshot.cn/v1 | kimi-k2.5 |
| Claude | https://api.anthropic.com/v1 | claude-3-opus |
| OpenAI | https://api.openai.com/v1 | gpt-4 |
| Ollama | http://localhost:11434/v1 | llama2 |

## 功能入口

- **导航页**: http://localhost:3000
- **实时对战**: http://localhost:3000/battle
- **卡组构筑**: http://localhost:3000/deck
- **数据管理**: http://localhost:3000/admin
